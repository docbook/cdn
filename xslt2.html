Welcome to the DocBook XSLT 2.0 Stylesheets CDN

Last update: 28 February 2018

This site hosts the numbered releases of the DocBook XSLT 2.0 stylesheets as well as the JavaScript, CSS, and other ancillary files referenced by those stylesheets.

You're welcome to leave the references in place (treating this site as a sort of DocBook CDN), or copy these assets to your local server. You can download them by cloning the repository.

These URIs may not be completely stable yet. I'm still trying to work out the best way to deal with versioning the ancillary files. I'm also trying to work out just how much disk space I can use at github for this purpose before they start to get cranky.

Starting with the 2.0.2 release, this site hosts versioned copies of the stylesheets and ancillary files.

Starting with the 2.1.2 release, this site hosts the resources and online files, but not the distributions. The distributions are available as releases on github or through Maven Central.
