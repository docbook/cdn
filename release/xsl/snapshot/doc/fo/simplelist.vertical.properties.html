simplelist.vertical.properties

simplelist.vertical.properties — Properties that apply to the layout table generated by simplelist in vertical mode.

Synopsis

<xsl:attribute-set name="simplelist.vertical.properties" use-attribute-sets="normal.para.spacing">
</xsl:attribute-set>

Description

Properties that apply to the fo:table generated by simplelist in vertical mode, that is, when @type="vert". You can change vertical spacing and other attributes.
