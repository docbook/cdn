Release Notes

This document summarizes the significant changes since the 2.0.4 release.

There's nothing significant about the order in which the changes are listed except that I usually try to put the big ones near the top..

  • Fixed bug where QNames were compared incorrectly in the schema extension preprocessor.

  • Added $syntax.highlight.map parameter to map verbatim language values.

  • Fixed bug where a customized locale was assumed to override everything in the locale.

  • Use $resource.slide in the plain-notes.xsl slides module (parallel to plain.xsl).
